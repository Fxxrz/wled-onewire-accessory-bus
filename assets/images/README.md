# Release Image Assets

The hardware images are edited release PNG cutouts with alpha transparency. The three UI images are anonymized AI-generated illustrative mockups, not captured evidence from a live WLED device.

## Files

- `lamp-lit-cutout.png`: assembled illuminated lamp, recommended hero image.
- `base-pogo-contacts-cutout.png`: base-side contact layout with center OneWire contact.
- `shade-contact-ring-cutout.png`: shade-side contact ring close-up.
- `shade-contact-ring-with-shade-cutout.png`: contact ring plus shade body.
- `open-base-electronics-cutout.png`: open base electronics/wiring reference.
- `wled-info-onewire-status.png`: illustrative WLED Info UI mockup.
- `wled-usermod-settings-onewire.png`: illustrative Usermod settings UI mockup.
- `onewire-manager-profile.png`: illustrative OneWire manager UI mockup.

## Notes

- The original HEIC files and intermediate review PNGs are intentionally not included.
- UI mockups use synthetic anonymized values. They are not protocol test vectors and must not be used as copy/paste ROM examples.
- The open electronics view is useful for technical documentation, but it should not be the first image in a public announcement.
- Re-run a visual privacy check before adding future photos, especially for labels, reflections, network names, serial numbers, or workspace details.
